# woocommerce-product-shortcode
A shortcode for displaying woocommerce product info

V1.0 Displays the basic woocommerce product info:
  - Title
  - Image
  - Regular Price
  - Sale Price
  - Availbility
  - Stock Status
  - Weight
  - Dimensions
    - Height
    - Width
    - Length
  - Long Desctiption
  - Short Description
  - Tags
  - Buy It Now Button
